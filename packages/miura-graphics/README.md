# `@miura/miura-graphics` (Coming Soon)

A library for 2D/3D rendering and animations within the miura framework.

## Features

- **High-Performance Rendering**: Optimized for performance, leveraging WebGL and WebGPU for hardware-accelerated graphics.
- **Declarative API**: Define complex scenes and animations using a simple, declarative API that integrates seamlessly with miura's reactivity system.
- **Component-Based Architecture**: Build reusable, component-based graphics that can be easily composed and managed.
